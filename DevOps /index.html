Frequently asked questions

How do I deploy this?

Save this page as index.html and upload it to any static host (GitHub Pages, Netlify, Vercel, Cloudflare Pages, or an S3 bucket with static hosting). That's it.

Where do I change colors and fonts?

Tweak CSS variables at the top of the stylesheet (:root). Colors like --brand and --brand-2 control the accent palette across the site.

Can I add more pages?

Yes. Duplicate this file and change the content. Update the nav links to point to your new files (e.g., about.html, services.html).
